# To-Do App

A full-stack task management application built with Next.js, TypeScript, Supabase, and PostgreSQL.

## Features

- Add, edit, delete, and complete tasks
- Search and filter tasks
- Due dates and priority levels
- Categories (Study, Work, Personal)
- Clear completed tasks
- Dark/Light mode
- Drag and drop reordering
- User authentication
- PostgreSQL database with Supabase

## Tech Stack

- Next.js
- TypeScript
- Supabase
- PostgreSQL
- Tailwind CSS
